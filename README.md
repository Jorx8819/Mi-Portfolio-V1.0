# 🖥 PORTFOLIO DE JORGE F. M.

---

## 🔹 Descripción del proyecto
Mi portfolio personal es una **experiencia interactiva** que combina **diseño moderno, animaciones tipo videojuegos y presentación de habilidades técnicas**.  
Está pensado para mostrar mis competencias en **frontend, desarrollo 3D, electrónica de potencia y software** de manera **impactante y seductora**.

## 🎮 Funcionalidades principales
- 🌗 **Modo Dark / Light**: cambio de tema dinámico con animaciones suaves.
- 💨 **Capa de fondo animada**: partículas de ceniza (`AshParticles`) y fuego degradado.
- 🕹 **HUD lateral**: estadísticas, logros y perfil con foto circular y efectos de escaneo.
- 🏗 **Sección de trayectoria profesional**: visualización tipo timeline con iconos.
- 🖥 **Tech Stack**: lenguajes y entornos IDE mostrados en tarjetas interactivas.
- 📂 **Proyectos / Repositorios**: enlaces a GitHub con estilo de tarjetas y hover effects.
- 🔗 **Footer con contacto**: botones para LinkedIn, GitHub y abrir formulario modal.
- ✉️ **Formulario de contacto modal**: envío a Formspree, con estado de envío y animaciones.
- 🖱 **Scroll & Cursor avanzado**: barra de progreso animada y cursor tipo crosshair.

## 🚀 Personalización
El portfolio es **altamente personalizable**:

- 🎨 **Temas y colores**: objeto `THEMES` (`dark` / `light`) controla fondos, glow y acentos.  
- 💨 **Efectos visuales**: partículas y animaciones mediante `AshParticles`.  
- 🏅 **Estadísticas y logros**: arrays `STATS` y `TRAYECTORIA`.  
- 🖥 **Tech Stack y proyectos**: arrays `TECH_STACK` y `PROJECTS` → se reflejan automáticamente en la UI.  
- 🖋 **Tipografía y gradientes animados**: clases de Tailwind y keyframes personalizados.
